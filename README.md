# MODULO1_PROYECTO_STREAMLIT

## Descripción

Aplicación interactiva desarrollada en Streamlit como proyecto aplicado del Módulo 1 de la Especialización en Python for Analytics.

La aplicación integra los conceptos fundamentales de programación aprendidos durante el módulo:

- Variables
- Condicionales
- Listas y diccionarios
- Bucles
- Funciones
- Programación funcional (map y lambda)
- Programación Orientada a Objetos (POO)

## Estructura del Proyecto

- `app.py` → Archivo principal de la aplicación
- `requirements.txt` → Librerías necesarias para ejecutar el proyecto
- `README.md` → Documentación del proyecto

## Funcionalidades

### Menu
Presentación del proyecto, autor e información general.

### Ejercicio 1
Verificador de presupuesto utilizando variables y estructuras condicionales.

### Ejercicio 2
Registro de actividades financieras usando listas y diccionarios.

### Ejercicio 3
Cálculo de retorno esperado mediante funciones y programación funcional.

### Ejercicio 4
Modelado de actividades financieras utilizando Programación Orientada a Objetos.
